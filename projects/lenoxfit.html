Project Overview

The Lenox fit website was built using react and aims to be a fast and beautiful redesign to the company's old website. This was put together over the span of a few weeks and features integration with the Mindbody API for scheduling and user account features. The website itself features a simple home page showcasing awards, services, contact and pricing as well as a simple gallery page.
