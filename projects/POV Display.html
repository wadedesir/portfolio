Project Overview

A Persistence of Vision Display is 2D Display that spins around a circular path and shows an image by blinking individual leds. This project was coded in C++ and used a range of microcontrollers before being finalized on a particle photon. An infrared LED was used to measure the RPM and activate an interrupt for each individual revolution that then determines the image refresh rate. To retain the ability to spin without any wires getting in the way, a wireless power transfer module is attached to the bottom of the fan and on the motor shaft. The full project was designed with the assistance of CAD Software. Autodesk Inventor for the chassis and Eagle for the Printed Circuit Board.
